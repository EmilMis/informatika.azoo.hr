# informatika.azoo.hr
search all users \n

run dataload.py to load latest users, takes up to 20 minutes to load all of them \n
`python3 dataload.py` \n
every user has: \n

|     Name     |  Registration Date  |     Description    |
|:------------:|:-------------------:|:------------------:|
| `User.name`  |    `User.reg_date`  | `User.description` |

to get all info, use \n
`user.info()`

have fun \n
