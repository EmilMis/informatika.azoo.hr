# informatika.azoo.hr
search all users

run dataload.py to load latest users, it takes up to 20 minutes to load all of them
`python3 dataload.py`
every user has:

|     Name     |  Registration Date  |     Description    |
|:------------:|:-------------------:|:------------------:|
| `User.name`  |    `User.reg_date`  | `User.description` |

to get all info, use
`user.info()`
have fun
